# Gcad Website

Official website for Gcad, built with React, Vite, and Shadcn UI.

## Getting started

> **Prerequisites:**
> The following steps require [NodeJS](https://nodejs.org/en/) to be installed on your system.

To get started with your project, you'll first need to install the dependencies with:

```bash
npm install
```

Then, you'll be able to run a development version of the project with:

```bash
npm run dev
```

After a few seconds, your project should be accessible at:
[http://localhost:5173/](http://localhost:5173/)

## Deployment

To build the project for release:

```bash
npm run build
```
